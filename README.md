# Time Tracker

An application to track your work time ⏰

## 🏭 Developing

👥 Clone the repository

```bash
git clone <Link>
```

🔽 Download and install the dependencies

```bash
npm install
```

💹 Start local developement server as electron program

```bash
npm run dev:electron
```

## 🧪 Testing

```bash
npm run test
```
